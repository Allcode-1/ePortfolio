# Frontend

React + TypeScript + Vite application for ePortfolio.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Environment

Copy:

```bash
cp .env.example .env
```

Required:

- `VITE_CLERK_PUBLISHABLE_KEY`
- `VITE_API_BASE_URL` (default `/api`)

For full project setup and Docker deployment see root [README](../README.md).
